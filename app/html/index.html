Automated Benchmarking Pipeline

All architectures were deployed and validated through a single automated pipeline, ensuring consistent, reproducible results.

  1. Automate Unit testing with Pyest.
  2. Build and upload images to AWS ECR.
  3. Provision AWS infrastructure with Terraform.
  4. Validate deployment with smoke tests .
  5. Execute automated load testing.
  6. Clean up all architecture to reduce cost.
