GitHub Actions CI/CD

Implement a branching strategy with GitHub Environments (dev/stage/prod) to manage and isolate multiple environments.

Automate unit tests and lint checks on every PR via GitHub Actions to ensure code quality.

Integrate and visualize performance testing with Grafana K6 to enforce API performance thresholds (error rate ≤ 0.01; p95 latency < 500 ms).

Jenkins CI/CD

Use JCasC (Jenkins Configuration as Code) to deliver consistent configurations and accelerate Jenkins provisioning. (e.g., automate plugin installation)

Leverage Jenkins Job DSL (Domain Specific Languages) with GitHub to codify and auto-provision jobs from the repo.

Serverless API & IaC

Serverless API on AWS

API Architecture (AWS)

  • S3 + DynamoDB + Lamdba + API Gateway + CloudFront + Cloudflare DNS

    • Infrastructure as Code (Terraform)

  • Configure S3 remote backend to lock resource state
  • Use tfvars for reusable configuration
  • Modularize functionalities to enable reuse
  • Dynamize resource creation with for_each statement

    • REST API Optimization

    Optimize REST API(AWS) performance

  • CloudFront: edge caching + response compression
  • API Gateway: method level caching + resource query string caching

    • Improve API performance

  • +500% higher peak RPS (70 reqs/s → 430 reqs/s)
  • -60% lower latency (264 ms → 105 ms)
  • -75% lower error rate (0.04 → 0.01)
